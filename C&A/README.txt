# פרויקט תנ״ך: קין והבל

נושא מרכזי: בחירה מוסרית ברגע של קנאה.

## מבנה הקבצים
- index.html
- style.css
- script.js

## איך לפתוח
1. לפתוח את התיקייה במחשב.
2. ללחוץ פעמיים על index.html.
3. מומלץ לפתוח גם דרך VS Code עם Live Server.

## מה יש באתר
- הסבר על סיפור קין והבל מתוך בראשית ד׳.
- רעיון מרכזי ברור: האחריות של האדם לרגשות שלו לפני שהם הופכים למעשה.
- פרשנות יצירתית סביב המשפט “לפתח חטאת רובץ”.
- אינטראקציות ב-JavaScript:
  - בחירת תגובת קין.
  - מד רגשי מתפתח.
  - אקורדיון שאלות למחשבה.
  - כפתור סיכום.
- עיצוב RTL מלא בעברית.
